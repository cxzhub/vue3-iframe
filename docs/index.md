# Props

| Props         | description                        | type     | default |
| ------------- | ---------------------------------- | -------- | ------- |
| width         | width                              | string   | 'auto'  |
| maxHeight     | max height                         | string   | 'auto'  |
| height        | height                             | string   | 'auto'  |
| inheritStyles | whether to inherit external styles | boolean  | false   |
| styles        | style array                        | string[] | []      |
